
SPI Master Function Library
***************************

scope:
   General Use

description:
   SPI Master Function Library tested for operation up to 25MHz SCLK
   frequency and all four SPI modes

keywords:
   SPI, Flash

boards:
   XK-1A, XK-SKC-L2

This is a function library for applications needing to implement SPI
master functionality. All four SPI modes are supported at clock speeds
up to 25MHz.
